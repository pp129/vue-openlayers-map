# v-vector

矢量图层组件，支持点、线、面等多种要素类型。

```js
import { VMap, VVector, VTile } from "v-ol-map";
```

## 默认示例

> 使用 features 属性添加矢量要素

<Canvas of={VVector.Default} />

## GeoJSON 基础示例

> 使用 GeoJSON 数据生成要素，支持要素级别样式(优先级高于图层样式)和图层统一样式。
>
> 直接使用 GeoJSON 很方便，**但 GeoJSON 标准不支持 Circle 几何类型。**

<Canvas of={VVector.GeoJsonBasic} />
